# Conversational Agents Looker Block

This Looker block provides a set of dashboards and explores for visualizing the structure and configuration of your Dialogflow CX agents.

## Dashboards

- **Agent Overview**: Provides a high-level overview of your agents, including the number of flows, pages, and intents.
- **Flow Details**: Allows you to drill down into the details of a specific flow, including its pages, routes, and transitions.
- **Intent Details**: Provides a detailed view of a specific intent, including its training phrases and parameters.

## How to Use

To use this block, you'll need to have a Looker instance with a connection to your BigQuery data. You can then install this block from the Looker Marketplace or by copying the files into your Looker project.
